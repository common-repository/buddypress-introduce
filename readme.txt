== Description ==

This plugin allows you to introduce your friends in BuddyPress

== Installation ==
1. Upload the files to wp-content/plugins/buddypress-introduce
2. Activate the plugin in your admin backend

That's it ... enjoy!

== Frequently Asked Questions == 
= What are the minimum requirements? =

That would be:
WordPress 3.1
BuddyPress 1.2.6
PHP 5

== Upgrade Notice ==

1. Deactivate BuddyPress Introduce
2. Backup any custom image, stylesheet and language files
3. Upload all files via FTP to wp-content/plugins/buddypress-introduce
4. Restore custom language, stylesheets and image files
5. Activate BuddyPress Introduce
